- Learn a Move Quiz - [Live](https://jackindisguise.github.io/pokemon-quiz/quiz-learn-move.html)

## Data + Assets Notes

- Quiz settings and score history are stored in browser `localStorage`.
- Pokemon image paths in quiz pages now use `.webp`.
